## ポール取り付け方式（確定）

**Uボルト方式（市販品）**

| 部品 | 仕様 | 状態 |
|---|---|---|
| Uボルト | M6、P=42.5mm、E=25.5mm、L=54mm | **購入済み** × 2本 |
| Uプレート | 対応品 | **購入済み** × 2枚 |
| M6ナット | 標準（対辺10mm、高さ5mm）| 要購入 × 8個 |
| M6ワッシャー | 標準 | 要購入 × 8個 |
| ゴム板 | 厚さ0.75mm（φ35+0.75×2=φ36.5mm） | 要調整 |
| M4ボルト+ナット | Bowl固定用 | 要購入 × 4組 |

### 組み立て手順

1. BackPlate凹面をポール（φ35mm）に当てる
2. Uボルト上（Z=+20mm）をポールに巻きつける
3. Uボルト足をBackPlateの穴（Z=±17mm）に通す
4. 足先端がボウル内側（Y≈-44mm）に出る
5. ボウル内側からナット+ワッシャーで締め付け
6. M4×4本でBackPlateをBowl背面に固定
7. 余分な足先端を現場でカット（約35mm）

### 重要: ポール位置

- ゴムを巻かないと内径φ36.5mmにφ35mmパイプが若干ゆるい
- ゴム0.75mm厚（片側）= 35+1.5=36.5mm でピッタリ
- またはゴム1mmのまま締め付け力で固定（Uボルト金属なので撓む）

## Bowl・Spider 分割方針（設計確定、**未実装**）

### 分割方式

- **Bowl・Spider 共に X=0 と Z=0 で4分割**（同一分割面）
- 同じ分割面にすることで「Bowl-L + Spider-L」を先に組んでから接合できる

### ダブルステップジョイント

```
 ￣￣￣|        ＿|￣￣￣
 ＿＿|￣        |＿＿＿
 （凸側）       （凹側）
```

| パーツ | 段数 | 段差寸法 | 幅 | クリアランス |
|---|---|---|---|---|
| Bowl（壁厚3mm）| 1段 | 1.5mm | 12mm | 0.2mm |
| Spider（リム10mm）| **2段** | 2mm×2 | 12mm | 0.2mm |

### 固定方法

- M3ネジ：Bowl側2本 + Spider側不要（リムが薄いため）
- 接着剤：永久接合（ステップ接触面に塗布）

### 接合部のネジ穴配置

**問題なし**（既に解決済み）：
- M3タッピング穴（Bowl固定用）: 45°/135°/225°/315°
- 分割線: 0°/90°/180°/270°
- → 45°ずれているのでネジ穴が分割線上に来ない ✓

### 組み立て順序（案1）

1. Bowl_LF を Spider_LF のU字溝に嵌める（左前半体）
2. Bowl_RF を Spider_RF のU字溝に嵌める（右前半体）
3. Bowl_LB を Spider_LB のU字溝に嵌める（左後半体）
4. Bowl_RB を Spider_RB のU字溝に嵌める（右後半体）
5. 左前+右前をX=0ダブルステップで合わせる
6. 左後+右後をX=0ダブルステップで合わせる
7. 前+後をZ=0ダブルステップで合わせる
8. M3ネジ締め + 接着剤（永久固定）
9. SpiderからBowlへM3タッピング×4本
